# Dog Kennel

Intro to OOP assignment. Two classes:

**Dog.java** — models a dog with name and weight. Has a `toKgs()` method that converts pounds to kilograms (x 0.453592). Uses encapsulation with private fields and public getters/setters.

**Kennel.java** — driver program that creates two dogs from user input, prints their info, and calculates the average weight in kg.

### How to run
```bash
javac Dog.java Kennel.java
java Kennel
```

### Concepts
Classes and objects, constructors, encapsulation, getters/setters, toString()
