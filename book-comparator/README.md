# Book Comparator

Book class that supports comparing two books by their page count. Uses a `compareTo` method that returns 1, 0, or -1 (similar to Java's Comparable interface but implemented manually).

TestBooks creates two book objects and compares them, then calculates the average page count.

### How to run
```bash
javac Book.java TestBooks.java
java TestBooks
```

### Concepts
Object comparison, compareTo pattern, constructors, toString()
